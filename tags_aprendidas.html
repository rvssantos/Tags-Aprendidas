Cabeçalho

Poderá ser elaborado através de títulos, que são representado com a letra "h" minúscula acrescido de um numeral e indo do h1 ao h6. Em síntese, quanto menor o número que acompanha a letra h, maior o título.

    h1 = Título Maior

    h2 = Vai diminuindo

    h3 = e diminuindo

    h4 = continua diminuindo
    h5 = menor
    h6 = o menor título

Parágrafo

Os parágrafos HTML são definidos com a tag: "< p >". Nele, podemos escrever o texto a que deverá ser exibido na página HTML.

Listas

As listas HTML permitem que os desenvolvedores da web agrupem um conjunto de itens relacionados em listas.

Poderá ser uma lista não ordenada, que começa com a "< ul >". Cada item da lista começa com a "< li >"". Essas listas serão delimitadas com marcadores.

Ou poderá ser uma lista ordenada, que começa com a "< ol >". Cada item da lista começa com a "< li >". Os itens da lista serão marcados com números

Demais Tags

"mark" destaca uma parte do texto.

"strong" negrita uma palavra.
